📖 처음 사용자를 위한 원격 연동 가이드

  • 1단계: 시스템 베이스 템플릿 사본 저장
    이 도구를 이용하려면 규격화된 데이터 시트 양식이 필요합니다. 아래 링크를 눌러 시트가 열리면 상단 메뉴의 [파일] ➔ [사본 만들기]를 눌러 본인의 구글 드라이브에 저장해 주세요.
    👉 [생두 관리 스프레드시트 템플릿 열기 🔗]
  • 2단계: 나만의 시트 고유 ID 확인
    사본으로 저장된 개인 스프레드시트의 인터넷 주소창을 확인합니다.
    주소창 고유 경로 중 /spreadsheets/d/ 와 /edit 사이에 위치한 영문 숫자 문자열이 회원님의 시트 ID입니다.
  • 3단계: 로그인 후 ID 기입 연동 완료
    상단의 [구글 계정으로 연동하기] 버튼을 통해 로그인하면 개인 시트 ID를 입력하는 제어창이 나타납니다. 복사한 고유 ID를 붙여넣고 저장하면 완료됩니다.
    👉 [도구 상세 사용설명서 매뉴얼 보기 🔗]
  • ⚠️ 중요: 'Google에서 확인하지 않은 앱' 경고창 대처법
    처음 계정 연동을 시도할 때 해당 경고창이 나타나는 경우, 당황하지 마시고 화면 왼쪽 아래의 [고급] 문구를 클릭한 후, 활성화되는 [생두 관리 데이터베이스로 이동(안전하지 않음)]을 선택해 주시면 정상 진행됩니다.
